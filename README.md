# GLOBAL-RESTAURANT-ANALYSIS-INTELLIGENCE-PLATFORM
Power BI solution delivering real-time analytics across 51K+ restaurant records using pure DAX measures, Power Query ETL, slicers, maps, matrices, line charts, KPIs, and visual panes.
Used the Global_Restaurant_Data and created Some PowerBI Dashboards using the BI's Abilities.
Strategic opportunity scoring through Growth_Score, Potential_Revenue, Competition_Index for unserstanding the insights.
Live slicer-driven insights for restaurant chain expansion decisions.
